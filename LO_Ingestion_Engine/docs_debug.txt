Diagnostic code guide:

D2Q: Tests knowledge of the correct method for converting a decimal number to quinary (repeated division by 5, reading remainders from bottom to top)

DQC: Tests the ability to correctly carry out the decimal to quinary conversion and arrive at the right quinary result

Q2D: Tests the ability to apply the positional value method to convert a quinary number to its decimal equivalent

BQC: Tests the ability to convert a binary number to quinary via the decimal intermediate step
